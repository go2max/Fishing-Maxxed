# Fishing Maxxed Release Notes

## 1.0.0 internal / production candidate

Fishing Maxxed is a private offline catch journal for anglers.

Initial release includes:

- Photo-based catch capture
- Known-reference length estimate
- Manual no-photo catch logging
- Draggable fish/reference measurement handles
- Measurement confidence and uncertainty labels
- Manual species confirmation
- Private local catch journal
- Edit notes and delete saved records
- Default CSV export that redacts exact coordinates
- Device-local leaderboard for personal bests
- Non-authoritative regulation guidance that directs users to official rules

Notes:

- Exact coordinates are private by default.
- No account or cloud leaderboard is included in this release.
- Regulation output cannot authorize keeping a fish. Always check official regulations.
