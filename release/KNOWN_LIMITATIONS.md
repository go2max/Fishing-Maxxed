# Known Limitations

Fishing Maxxed is intentionally conservative until device testing and regulation validation are complete.

## Measurement

- Length estimates depend on reference placement, image angle, lighting, and handle placement.
- Physical fixture validation is still required before release readiness can improve.
- Manual length entries are treated as low-confidence fallback records.

## Regulations

- Bundled rules are a non-authoritative fixture.
- The app cannot authorize Keeper with bundled data.
- Users must check official regulations before keeping a fish.
- No daily-current California or worldwide regulation coverage is claimed.

## Privacy

- Records are local to the device.
- Default CSV export excludes exact coordinates and private photo paths.
- Private coordinates can only be exported through explicit code-level opt-in.

## Sync And Leaderboards

- There is no cloud sync.
- There is no account system.
- Leaderboards are local to the current device.

## Release

- Signed APK/AAB artifacts have not been verified in this environment.
- Play support contact details still need to be finalized.
- Samsung S22 Ultra capture-to-reopen QA is still required.
