# Fishing Maxxed

## Short description

Measure catches, confirm species, and keep a private on-device fishing journal.

## Full description

Fishing Maxxed provides camera capture, known-reference length estimates with correction handles and uncertainty, a searchable species catalog, private catch records, broad-region exports, and personal on-device leaderboards. Exact coordinates stay private and are excluded from default exports.

Regulation checks are conservative. Unsupported, stale, or ambiguous rules return Unable to verify and never authorize Keeper. Always verify official regulations before keeping a fish.
