# Data Safety Draft

Status: draft for a blocked internal-testing candidate.

Current artifact assessment:

- Data collected: none observed beyond local app-private preference storage.
- Data shared: none observed.
- Data sold: no.
- Processing model: on-device only.
- User deletion: uninstall removes app-private local state.
- Encryption in transit: not applicable because no network features are implemented in this build.

Required revision before any upload:

- Reassess this draft if future versions add photo retention, location storage, redacted exports, analytics, or cloud sync.
